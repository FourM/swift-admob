# First Native App using

## Get Started
1. Prepare cocoapods 
```
sudo gem install cocoapods
```

```
sudo gem install -n /usr/local/bin cocoapods
```

```
pod setup
```

2. Clone app
```
git clone git@github.com:FourM/swiftui-first.git
```

3. Install dependencies
```
cd /path/to/project
pod install --no-repo-update
```
